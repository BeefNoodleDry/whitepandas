## Typography

### Primary — Headings

- **Typeface:** Recoleta Bold
- **Fallback:** Cooper Black
- **Line height:** 0.80 - 1.1 (larger fonts use smaller line-height)

### Secondary — Subheadings & Body

- **Typeface:** Glacial Indifference Regular / Bold
- **Fallback:** Arial Regular / Bold
- **Line height:** 1.3 minimum (body copy)

### Letter Spacing

| Context                              | Spacing |
| ------------------------------------ | ------- |
| Body text (including bold)           | 0       |
| Subheadings, meta info, secondary buttons | 25%     |
| Primary button text (uppercase)      | 50%     |

## UI Elements

### Buttons

**Primary CTA:**
- Background: `#E1241B` (RP Red)
- Text: white, uppercase, letter-spacing 50%
- Shape: rounded (pill)
- Hover: `#222222` (Charcoal)
- Inactive: `#C8C8C8`

**Primary Button (small):**
- Background: `#E1241B` or `#222222`
- Text: white, uppercase
- Hover: fills with Charcoal / darkens

**Secondary Button (on coloured bg):**
- Border: white, no fill
- Text: white / BG Red
- Hover: fills with Charcoal

**Secondary Button (on white bg):**
- Border: Charcoal, no fill
- Text: Charcoal
- Hover: fills with Charcoal, text turns white

### Hyperlinks

- Default: BG Red `#E5453C`
- Hover: underline
- Disabled: greyed out
- Visited: BG Red (slightly muted)

### Corners

- **Containers / cards:** sharp corners (no border-radius)
- **Buttons only:** rounded corners (pill shape)

### Borders & Shadows

- **Borders:** grey, to separate same-colour elements from background
- **Hard shadow:** Beige `#FCE9E8`, offset 45%, opacity 100%, blur 0%, size 4pt

## Icons

- Style: sharp corners, solid fill
- Source: Font Awesome Sharp Solid
- Can be placed bare or inside circle / irregular polygon containers
- Use primary palette + RP Red shades; prioritise high contrast

---

## Design Elements

### Stickers

- Max 2 per design/section
- Shapes: circles, simple squiggles/doodles, stars
- Can be scaled, rotated, cropped freely
- Colours: primary palette + RP Red shades

### Layering

- Use irregular polygons or circles behind/around images
- Image treatments: frame in polygon, circle behind, cutout with outline

### Typography Treatments (headings only, Recoleta)

- Background layer: solid fill colour
- Foreground layer: thin outline, no fill, -45deg angle with slight offset
- Use different colours for foreground vs background

---

## Best Practices

- Clarity first — nothing obscures text legibility
- Embrace whitespace for hierarchy and breathing room
- Strive for visual balance; every element should serve a purpose
- Max 2 stickers per design/section
